# Tours in La Paz-Bolivia

Este es un pagina web que permite obtener una experiencia agradable a la hora de visitar nuestro pais Bolivia, en particular el departamento de La Paz.

## Instrucciones

1. Para correr este proyecto abre una terminal en la carpeta raiz y ejecute los siguientes comandos:
```
npm install 
npm run dev
```
